Detailed Description

Waveform playback configuration.

This defines the configuration parameter structure for initializing and using the Waveform Memory Decoder SW module (that is used during waveform sequence playbacks). It is part of the Adapter's configuration parameter structure.

See also
ad_haptic_controller_conf_t, ad_haptic_play_wm_sequence()

Field Documentation

bool ad_haptic_wm_conf_t::accel_en

Indicates whether or not the waveform sequence amplitude is in unsigned (true, positive values only) or signed format (false, positive and negative values). If false (signed format case), then the acceleration and braking is supposed to be encoded in the waveform sequence, so, in this case, Overdrive is not applicable.

const uint8_t* ad_haptic_wm_conf_t::data

Pointer to the byte array buffer in which the encoded waveform sequences are stored (waveform memory).

bool ad_haptic_wm_conf_t::timebase

If false, then the waveform timebase is AD_HAPTIC_WM_SEQUENCE_TIMEBASE * 4, otherwise it is AD_HAPTIC_WM_SEQUENCE_TIMEBASE.

Detailed Description

Haptic Low Level SW Driver configuration.

This defines the configuration parameter structure for initializing the Haptic Driver.

See also
hw_haptic_init()

Field Documentation

uint16_t haptic_config_t::duty_cycle_abs_max

maximum absolute duty cycle of the drive signal that is supported by the actuator (transient while accelerating/braking) (in unsigned Q1.15 fixed-point format (0-32768 corresponding to 0-100%)). It should be lower or equal to HW_HAPTIC_DREF_MAX.

uint16_t haptic_config_t::duty_cycle_nom_max

maximum nominal duty cycle of the drive signal that is supported by the actuator (steady state operation) (in unsigned Q1.15 fixed-point format (0-32768 corresponding to 0-100%)). It should be lower or equal to HW_HAPTIC_DREF_MAX and also lower or equal to duty_cycle_abs_max.

uint16_t haptic_config_t::half_period

half period of the drive signal in case of an LRA (should be set according to its nominal/initial resonant frequency) - or just half period of the interrupt requests in the ERM case (optional). If set, a value between 250 and 5000 (corresponding to a frequency range of 25-500Hz) is recommended (allowing enough time for the System to service the Haptic interrupt requests).

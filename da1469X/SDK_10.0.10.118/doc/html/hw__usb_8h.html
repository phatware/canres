void hw_usb_restore_int_mask_at_resume ( void  )

Restore interrupts masks after resume.

Remarks
On suspend, all interrupts are masked except for ALTEV interrupts. So, when getting out of suspend (VBUS falling or USB Reset or Resume interrupt pending), the masks must be removed before continuing.

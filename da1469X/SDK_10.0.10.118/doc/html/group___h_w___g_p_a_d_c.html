__STATIC_INLINE void hw_gpadc_select_diff_temp_sensor ( HW_GPADC_TEMP_SENSORS  sensor)

Selects on-chip temperature sensor.

Parameters
[in]sensoron-chip temperature sensor
Note
When temperature sensors are enabled (GP_ADC_DIFF_TEMP_EN=1), then: 0 = GND, 1 = sensor near radio, 2 = sensor near charger, 3 = sensor near bandgap. When temperature sensors are disabled (GP_ADC_DIFF_TEMP_EN=0), then: 0 = GND, 1 = Z, 2 = V(ntc) from charger, 3 = V(temp) from charger.
Users are advised NOT to use this API function, unless they know exactly what they are doing. In the general case, setting the gpadc->temp_sensor and calling hw_gpadc_init() or hw_gpadc_configure() is enough.

__STATIC_INLINE void hw_gpadc_set_oversampling ( uint8_t  n_samples)

Set oversampling.

With oversampling enabled multiple successive conversions will be executed and results are added together to increase effective number of bits in result.

Number of samples taken is 2n_samples. Valid values for n_samples are 0-7 thus at most 128 samples can be taken. In this case, 17bits of result are generated with the least significant bit being discarded.

Parameters
[in]n_samplesnumber of samples to be taken

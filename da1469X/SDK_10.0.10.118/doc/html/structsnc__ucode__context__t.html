uint32_t snc_ucode_context_t::cmd

Flag indicating a pending notification event from SYSCPU (CM33) to the uCode-Block (1: pending event)

uint32_t snc_ucode_context_t::flags

uCode flags as defined in SNC_UCODE_BLOCK_CMD enumeration bit0 - Command requesting CM33 control over a uCode-Block resource bit1 - Command requesting uCode-Block to be disabled

Link pointer to jump back to the previous uCode in the execution flow uCode flags as defined in SNC_UCODE_BLOCK_FLAG enumeration bit0 - Flag indicating SNC control over a uCode-Block resource bit1 - Flag indicating uCode-Block-is-running

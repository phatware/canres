enum HW_TIMER_GPIO

GPIOs for timer trigger.

In HW_TIMER_MODE_TIMER mode use to select which GPIO will trigger capture time. In HW_TIMER_MODE_ONESHOT mode use to select which GPIO will trigger the programmable pulse. HW_TIMER and HW_TIMER3 are mapped to GPIO Port 0 where HW_TIMER2 and HW_TIMER4 are mapped to GPIO Port 1.

#define SNC_CLR_BIT (   addr,
  bit_pos 
)    _SNC_CLR_BIT(addr, bit_pos)

Macro used in SNC context to clear a bit.

It clears a bit in a Register/RAM address.

For a given RAM address, the addressing mode can be either direct (i.e. da()) or indirect (i.e. ia()). For a given Register address, the addressing mode is always direct (i.e. da()).

Parameters
[in]addr(uint32_t*: use da() or ia()) the Register/RAM address at which the bit is cleared
[in]bit_pos(uint32_t: build-time-only value) the bit position

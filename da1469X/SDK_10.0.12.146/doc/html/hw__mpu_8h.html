Detailed Description

Definition of API for the Memory Protection Unit (MPU) Low Level Driver.

The MPU is an optional ARM CM33 feature supported in DA14yyx SoC families that enables protecting loosely defined regions of system RAM memory through enforcing privilege and access rules per region. All MPU LLD terminology in based on the ARM CM33 nomenclature.

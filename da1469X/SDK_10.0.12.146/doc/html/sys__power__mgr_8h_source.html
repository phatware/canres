312 /*
313  * \brief Put the system to idle or sleep or block in a WFI() waiting for the next tick, if neither
314  * idle nor sleep is possible.
315  *
316  * \detail Puts the system to idle or sleep, if possible. If an exit-from-idle or a wake-up is
317  * needed, it programs the Timer1 to generate an interrupt after the specified idle or sleep
318  * period. Else, the system stays forever in idle or sleep mode.
319  * If neither idle nor sleep is possible, it blocks in a WFI() call waiting for the next
320  * (already programmed) OS tick to hit.
321  *
322  * \param[in] low_power_periods The number of (prescaled) low power clock periods the OS will be
323  * idle. If it is 0 then the OS indicates that it can block forever waiting for an
324  * external event. If the system goes to sleep, then it can wake up only from an external
325  * event in this case.
326  *
327  * \warning Must be called with interrupts disabled!
328  *
329  */
